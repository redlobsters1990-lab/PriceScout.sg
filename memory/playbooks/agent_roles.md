# Agent Roles

## chief_of_staff
- executive brief
- final internal selector for which approved opportunities become pages
- owns end-of-day memory promotion review for approved/rejected opportunities and strategic patterns

## orchestrator
- central dispatcher and state manager
- delegates work and consolidates outputs upward

## market_intel
- mines query patterns, datasets, SERP structures, monetization signals, weak competitors, and SERP gaps
- rejects weak opportunities aggressively

## research
- scores approved opportunities with a hard rubric
- requires a clear publishable blueprint

## gatekeeper
- rejects weak buyer intent, unclear monetization, dominated SERPs, or weak blueprints

## planner
- turns approved opportunities into site/page architecture

## builder
- renders structured pages from approved plans

## publisher
- QA, local-first publish, and SEO deployment

## revenue_ops
- affiliate-first monetization, optional lead capture, ads later

## growth_ops
- post-publish maintenance, freshness, ranking/listing/price updates
